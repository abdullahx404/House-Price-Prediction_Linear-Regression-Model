# House Price Prediction using Linear Regression

## Requirements

- Python 3.11 or newer
- Git
- Kaggle Account (Optional)
- Kaggle API Token (Optional)

## Installation

Clone the repository

```bash
git clone <repo-url>
cd House-Price-Prediction_Linear-Regression-Model
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

Windows:

```cmd
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Verify installation

```bash
python --version
pip --version
kaggle --version
```

## Dataset

1. Use the dataset provided in the repo [OR]

2. Download the dataset from Kaggle

```bash
kaggle datasets download -d yasserh/housing-prices-dataset -p data
```

Extract the ZIP file into the `data` folder.
